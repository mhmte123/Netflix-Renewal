# 로컬 자막 파일

주요 작품의 한국어 자막을 직접 제공해 **풀 커스텀 자막**(크기·폰트·색상·배경·그림자)을 보장하는 폴더입니다.

## 사용 방법

1. 자막을 넣을 영상의 유튜브 videoId 확인
   - 영상 재생 시 브라우저 콘솔에 `[자막] video=XXXXXXXX ...` 로 출력됩니다.
2. 이 폴더에 `{videoId}.json` 파일 생성 (예: `LdezlX84py0.json`)
3. 형식은 `_example.json` 참고 — `start`/`end`는 초 단위

```json
[
  { "start": 1.2, "end": 3.8, "text": "첫 번째 자막 줄" },
  { "start": 4.0, "end": 7.5, "text": "두 번째 자막 줄" }
]
```

## 동작 방식

- 로컬 파일이 있으면 → 커스텀 자막 오버레이 (프로필 자막 설정 100% 반영)
- 없으면 → 유튜브 timedtext 시도 (현재 유튜브 측 차단으로 대부분 실패)
- 그래도 없으면 → 유튜브 내장 자막 폴백 (CC 토글 가능, 커스텀은 크기만)
